# renderer.gua
software 3D renderer


## Usage

### OS X
```shell
sh install_mac_sdl.sh 
make osx
./gua3d
```

### Windows (MinGW32)
```shell
build_win_mingw.bat
gua3d
```

### Linux
```shell
# Depends on libsdl1.2-dev. For Ubuntu: sudo apt-get install libsdl1.2-dev.
make linux
./gua3d
```
